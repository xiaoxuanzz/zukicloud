云存储 API 开发文档

概述

本API提供文件上传、下载、列表查询及管理功能，支持RESTful风格调用。

基础URL: http://your-domain/api/index.php

请求格式: JSON (部分接口为FormData)

响应格式: JSON (文件下载除外)

认证方式

普通用户 (API Key)

  • Header方式: X-API-KEY: <your_api_key>
  • URL参数方式: ?api_key=<your_api_key>

管理员 (Admin Key)

  • Header方式: X-Admin-Key: <your_admin_key>
  • URL参数方式: ?admin_key=<your_admin_key>

API Key存储在 api/keys.txt，Admin Key存储在 api/admin_key.txt

API 端点

1. 查询API状态

GET /api/index.php?action=status

认证: 无需认证

响应示例:

{
  "api_enabled": true,
  "version": "2.0"
}

2. 获取文件列表

GET /api/index.php?action=list&api_key=<key>[&path=<path>&recursive=false]

参数:

参数类型说明
pathstring子目录路径 (如: docs)
recursiveboolean是否递归列出子目录 (默认false)

认证: 需要API Key

响应示例:

{
  "path": "",
  "contents": [{
      "name": "PotPlayerMini64.reg",
      "path": "f78fe335885a81f6f592c83af51435bc",
      "type": "reg",
      "size": 31840,
      "mtime": 1778068540,
      "hash": "f78fe335885a81f6f592c83af51435bc"
  }],
  "count": 1
}

3. 上传文件

POST /api/index.php?action=upload&api_key=<key>

Content-Type: multipart/form-data

参数:

参数类型说明
fileFile要上传的文件 (必填)
deststring目标目录 (可选，如: docs/images)

认证: 需要API Key

响应示例 (新上传):

{
  "success": true,
  "exists": 0,
  "hash": "f78fe335885a81f6f592c83af51435bc",
  "name": "PotPlayerMini64.reg",
  "size": 31840,
  "type": "reg",
  "id": 72,
  "downurl": "http://localhost:8001/down.php/f78fe335885a81f6f592c83af51435bc.reg"
}

4. 下载文件

GET /api/index.php?action=file&path=<hash>&api_key=<key>

认证: 需要API Key

响应: 文件流 (Content-Disposition会设置为原始文件名)

5. 管理员操作

5.1 开关API

POST /api/index.php?action=admin_toggle&admin_key=<key>

Content-Type: application/json

Body:

{ "enabled": true }

响应示例:

{ "success": true, "api_enabled": true, "message": "API enabled" }

5.2 重新生成API密钥

POST /api/index.php?action=regenerate_key&admin_key=<key>

响应示例:

{ "success": true, "key": "newly_generated_key_here" }

cURL 测试示例

# 查询状态
curl "http://localhost:8001/api/index.php?action=status"

# 获取文件列表
curl -H "X-API-KEY: your_key" "http://localhost:8001/api/index.php?action=list"

# 上传文件
curl -X POST -H "X-API-KEY: your_key" -F "file=@/path/to/file.pdf" "http://localhost:8001/api/index.php?action=upload"

错误处理

HTTP状态码说明
200请求成功
400错误请求
401认证失败
403禁止访问
500服务器内部错误
